### Rapid Prototyping (draw.io)

To quickly create product prototypes:

**Step 1 — Download Templates**
Download the provided `.xml` or `.drawio` files from the attachments.

**Step 2 — Upload Templates to draw.io**
1. Open the draw.io editor
2. Go to **File → Import**
3. Upload the template files
4. The new components will be added to your left-side library panel

**Step 3 — Start Prototyping**
- Drag-and-drop UI elements onto the canvas
- Customize styling via the Properties panel
- Export screens as PNG, PDF, or embed links
- Create flows using connectors
